# Animated Pip-Boy Light
<p align="left"> <img height="350px" src="https://staticdelivery.nexusmods.com/mods/1151/images/102694/102694-1773963877-1386304138.png"> </p>

## Overview
Adds a unique first-person animation for toggling the Pip-Boy light.

## Requirements
- [F4SE](https://www.nexusmods.com/fallout4/mods/42147)
- [Garden of Eden Papyrus Extender](https://www.nexusmods.com/fallout4/mods/74160)

## Installation
Installs like any other mod, manually or using a mod manager.
